# Flappy Bird played by Reinforcement Learning Algorithms


## Algorithms
- Asynchronous Actor-Critic Agents (A3C)
- Deep Q-Network

## Installation Dependencies:
* Python 3.5
* pygame
* Keras 2.0
* scikit-image
* TensorFlow
* OpenCV-Python
* h5py

## How to Run?
```
git clone https://github.com/jgresc/FlappyBird-GameAgent.git
cd RLGameAgents
python run.py
```
